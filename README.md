# PayMind
PayMind is an AI-powered payment agent for managing real estate transactions using USDC on the Arc blockchain. Users can issue natural language commands like “Pay my rent on the 1st of every month,” and the agent will autonomously execute the payment using Circle’s wallet infrastructure.
# PayMind System

This system consists of three main components:

## 1. API (@p-karmelita/API)
This project is the API, providing all the necessary endpoints for data access, system security, and smooth integration across the services.
- **Primary Function:** Manages operations between the front end and back end.
- **Project Link:** [p-karmelita/API](https://github.com/p-karmelita/API)

## 2. Front End (@SHAHBAZ-SHEIKH/AIPayMind)
This project is the user-facing front end, allowing users to interact with the system.
- **Primary Function:** Displays data and facilitates user actions and input.
- **Project Link:** [SHAHBAZ-SHEIKH/AIPayMind](https://github.com/SHAHBAZ-SHEIKH/AIPayMind)

## 3. Back End (@p-karmelita/PayMind)
This project is the backend, responsible for database management and core logic.
- **Primary Function:** Processes operations and manages data.
- **Project Link:** [p-karmelita/PayMind](https://github.com/p-karmelita/PayMind)

---

## How to Run

1. Install the required dependencies for each project as described in its respective README file.
2. Start the API first, then launch the back end, and finally, run the front end.
3. Make sure to configure the connections and endpoints between each part correctly.

PayMind is a comprehensive financial management solution powered by an easy-to-use interface, reliable API, and robust backend.
